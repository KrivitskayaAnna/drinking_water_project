# drinking_water_project
Exploration of data: https://data.world/arnholdinst/drinking-water-contaminations

Выполнен анализ поставщиков:
1) Поставщики каких стран представлены
2) Распределение по штатам
3) Сводные таблицы по нарушителям и насколько тяжелыми были нарушения
4) Какой поставщик нанес наибольший урон, а какой наименьший. Какого поставщика лучше выбрать
5) Анализ возможных загрязнений, какие бывают и какие опасны
6) Возможная связь геолокации поставщика и уровня загрязнения

Проект содержит:
1) Jupyter notebook
2) Power BI report
